# Chat HTML Code

A minimal static chat UI project containing HTML and CSS files for a simple chat page.

## Files

- `index.html` — main HTML page
- `style.css` — stylesheet for the page

## Description

This project is a small static web page demonstrating a chat UI layout. It's meant to be opened locally in a browser or served with a simple static file server.

## Preview locally

Option 1 — open directly in your browser:

- Double-click `index.html` or open it from your browser's File > Open dialog.

Option 2 — run a quick local server (recommended):

If you have Python 3 installed, run:

```bash
python -m http.server 8000
```

Then open http://localhost:8000/ in your browser.
